### Function #7: BMI Calculator
**Valid Inputs:**
- weight: `double` (Range: 1.0 to 1000.0 lbs)
- height: `double` (Range: 1.0 to 120.0 inches)

**Expected Output(s):**
- BMI Value: `double` (Formula: 703 * weight / height^2)
- Category: `String` (e.g., "Normal", "Overweight")

**Invalid Input Error Message:**
- "Error: Height and weight must be positive values."
